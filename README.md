# S&P 500 量化投資與機器學習研究專案

## 專案簡介
本專案旨在利用機器學習技術，針對 S&P 500 指數成分股進行量化分析與預測。內容涵蓋兩大核心實驗：
1. **回歸預測實驗**：利用 PCA 與 PLS 降維技術預測季度對數報酬率。
2. **分類預測實驗**：利用 Lasso 與 Ridge 正規化模型處理分類任務，並探討金融數據中的「樣本不平衡」與「高噪音」問題。

---

## 通用數據處理流程
為確保模型穩健性並避免統計陷阱，所有實驗均遵循以下準則：
* **數據源**：S&P 500 成分股 (2024/01/01 - 2025/12/31)。
* **缺失值處理**：以該季度中位數進行填補。
* **異常值處理**：針對 1%~99% 之資料進行縮尾 (Winsorization)。
* **前瞻偏差 (Look-ahead bias) 控制**：財報相關因子（如 ROE、成長率等）均採用至少 90 天前的最新數據。
* **特徵因子 (21 個)**：包含動能組（1W/1M/3M/6M/12M）、波動組（20D/60D 波動、下行波動）、流動性組（換手率、Amihud）及基本面組（ROE、營收/資產成長率、應計項目）。

---

## 實驗一：季度報酬率回歸分析 (Regression Analysis)

### 1. 研究目標
採用滾動執行 (Rolling Execution) 方式，預測下一季度的對數報酬率，探討不同預測準則下的模型泛化能力。

### 2. 模型架構
* **PCR (主成分回歸)**：分別根據 $k_{CV}$、$k_{AIC}$ 及 $k_{Onatski}$ 決定主成分數量。
* **PLS (偏最小二乘法)**：以 PLS 1-component 作為基準模型。

### 3. 關鍵發現
* **降維效果**：$k_{AIC}$ 與 $k_{Onatski}$ 選出的主成分數接近原始因子數，顯示因子間相關性低，降維意義有限，且存在過擬合風險。
* **預測力來源**：自 2025Q2 起，隨著財報因子（銷售與資產成長率）的數據完整度提升，模型 RMSE 出現顯著下降。
* **PLS 之優勢**：PLS1 的泛化能力在多數期間優於 PCR，暗示 20 個因子可被歸納為單一潛在因子 (Latent factor) 共同驅動市場。
